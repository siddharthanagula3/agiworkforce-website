## Email Automation Tools

### Gmail Filters & Labels (Free)

**Best For:** Basic organization

**Setup:**
1. Settings → Filters and Blocked Addresses
2. Create new filter
3. Define criteria (from, subject, has words)
4. Choose actions (label, archive, forward, delete)

**Example Rules:**
```
From: invoices@vendor.com
→ Label: "Invoices"
→ Star it
→ Forward to accounting@company.com

Subject: "newsletter"
→ Label: "Reading"
→ Skip inbox (archive)

From: boss@company.com
→ Star it
→ Mark as important
→ Never send to spam
```

**Time Saved:** 1 hour/week

### Outlook Rules (Free)

**Similar to Gmail Filters:**
1. Home → Rules → Manage Rules & Alerts
2. New Rule
3. Define conditions
4. Define actions

**Advanced:** Use Quick Steps for multi-action workflows

**Time Saved:** 1-2 hours/week

### Email Templates

**Gmail:**
Settings → Advanced → Templates → Enable

**Create Template:**
1. Compose email
2. More options (⋮) → Templates → Save draft as template

**Use Template:**
- Compose → More options → Templates → Insert template

**Outlook Quick Parts:**
Insert → Quick Parts → Save Selection

**Time Saved:** 3 hours/week

## Email Automation Best Practices

### Rule 1: Start with Organization

**Don't automate chaos.** Clean up first:

1. **Unsubscribe:** 50% of emails you don't need
2. **Archive:** Move old emails out of inbox
3. **Create Labels/Folders:** Clear categories
4. **Set Up Filters:** Auto-sort new emails

**Goal:** Inbox Zero daily

### Rule 2: Use Templates Wisely

**Good Uses:**
- Frequent questions
- Status updates
- Meeting scheduling
- Thank you notes

**Bad Uses:**
- Personalized responses
- Sensitive topics
- Complex issues

**Pro Tip:** Customize templates before sending

### Rule 3: Don't Over-Automate

**Keep Human:**
- First contact with clients
- Performance reviews
- Sensitive topics
- Complex negotiations

**Automate:**
- Routine updates
- Data transfer
- Organization
- Reminders

### Rule 4: Monitor and Adjust

**Weekly Review:**
- Check automation logs
- Verify correct routing
- Update rules as needed
- Remove unused automations

### Rule 5: Respect Recipients

**Never:**
- Spam people
- Auto-reply to all
- Send marketing without consent
- Ignore unsubscribes

**Always:**
- Provide value
- Allow opting out
- Be transparent about automation
- Maintain quality

## Common Mistakes to Avoid

**Mistake 1: No Review Process**
✗ Automations run blindly
✓ Review logs weekly

**Mistake 2: Too Generic**
✗ One-size-fits-all templates
✓ Personalize before sending

**Mistake 3: Over-Filtering**
✗ 100 filters, can't find anything
✓ 10-15 well-designed filters

**Mistake 4: No Backup**
✗ Automation deletes important email
✓ Archive, never delete automatically

**Mistake 5: Ignoring Spam Filters**
✗ Automated emails go to spam
✓ Test deliverability, follow best practices
